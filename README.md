This project is a Student Management System developed using the C programming language.
It is a menu-driven application designed to manage student records efficiently through basic operations such as adding, displaying, updating, and sorting student data.
The system allows users to input student details including name, roll number, and marks, which are stored dynamically in memory using pointers and realloc for flexible data handling.
Users can view all student records, calculate the average marks of the class, and search for a specific student using their roll number to update their details.
The program also includes functionality to calculate the percentage of a student based on their marks.
In addition to in-memory operations, this system supports file handling, allowing users to save student data to a file and load it later for continued use. 
The sorting feature enables arranging student records based on roll numbers or marks, making data organization more efficient.
The application is implemented using standard C libraries such as stdio.h and stdlib.h and demonstrates key programming concepts including structures, dynamic memory allocation, file handling, and modular programming through functions. 
This project is suitable for beginners to understand how real-world data management systems work in a console-based environment and serves as a foundational step toward more advanced software development.
